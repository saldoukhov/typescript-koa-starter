# TypeScript Koa Starter

Boilerplate for Koa and TypeScript applications.

## Getting Started
```sh
# Clone it
git clone https://github.com/saldoukhov/typescript-koa-starter.git

# Remove git
rm -rf .git && git init

# Make it your own
yarn init

# Install dependencies
yarn

# Running tests
yarn test

# Build and start server
yarn build && yarn start

```
